# Rubyemail

Ruby email parser - unmarshals an email message into message, body, bodyparts and headerfield objects

## Installation

Add this line to your application's Gemfile:

    gem 'rubyemail'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rubyemail
